# Audio Design Studios Website

Portfolio website for Karter G. Brown - Audio Design Studios

## Development Workflow

After making changes to HTML, CSS, or JS files:

1. **Update cache-busting versions** (forces browsers to reload files):
```bash
./update-version.sh
```

2. **Commit and push your changes**:
```bash
git add .
git commit -m "your message"
git push
```

This ensures browsers always load the latest version of your files instead of cached versions.
